MediaTek86 - Управление персоналом и отсутствиями

Описание проекта

Проект MediaTek86 представляет собой настольное приложение для управления персоналом и их отсутствиями в медиатеке. Приложение разработано на языке C# с использованием базы данных MySQL. Оно позволяет добавлять, редактировать и удалять записи о сотрудниках и их отсутствиях, а также предоставляет удобный интерфейс для взаимодействия с базой данных.

Цели проекта

  • Создание удобного интерфейса для управления данными о персонале.
  • Реализация функций добавления, редактирования и удаления записей.
  • Управление отсутствиями сотрудников с возможностью добавления и редактирования информации.
