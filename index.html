MediaTek86 - Gestion du personnel et des absences

Description du projet

Le projet MediaTek86 est une application de bureau pour la gestion du personnel et de leurs absences dans la médiathèque. L'application est développée en C# avec une base de données MySQL. Elle permet d'ajouter, de modifier et de supprimer des enregistrements sur les employés et leurs absences, tout en offrant une interface utilisateur conviviale pour interagir avec la base de données.

Objectifs du projet

  • Créer une interface conviviale pour gérer les données du personnel.
  • Implémenter des fonctionnalités pour ajouter, modifier et supprimer des enregistrements.
  • Gérer les absences des employés avec la possibilité d'ajouter et de modifier des informations.
